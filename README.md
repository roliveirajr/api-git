# Repository - API Git

## Pre-requisites

* Node - http://nodejs.org/

## Install

Clone this repo or download and unzip it.

## Quick Start

Enter the folder you cloned or downloaded, install dependencies and run `npm i`:

```shell
    $ cd api-git
    $ npm i
    $ gulp
```
Currently supported:

- LiveReload of assets HTTPS
- Stylus compilation
- CSS Minification
